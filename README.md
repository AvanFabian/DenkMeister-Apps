# Denkmeister
## Platform Pembelajaran bahasa Jerman

1. Clone repository:

    ```bash
    git clone https://github.com/AvanFabian/AppTebakGambar.git
    ```

2. Navigate to the project directory:

    ```bash
    cd AppTebakGambar
    ```

3. Install Flutter Dependencies:

    ```bash
    flutter pub get
    ```
